# Crypto Bot

Open-source система для мониторинга аномалий курса криптовалют с использованием машинного обучения и уведомлениями в Telegram

## О проекте
Проект разрабатывается в рамках учебных курсов по "Разработке Open Source" и "Информатике". Основная цель — создать автономный инструмент, который помогает трейдерам замечать резкие изменения на рынке без необходимости постоянно следить за графиками.

## Стек технологий
- **Backend:** Python 3.10+
- **Data Science:** Pandas
- **Database:** SQLite
- **Infrastructure:** Docker, Makefile
