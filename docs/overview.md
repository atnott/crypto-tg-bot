# Обзор проекта

Crypto Telegram Bot — система для мониторинга аномалий курса криптовалют
с уведомлениями в Telegram.

Проект собирает рыночные данные с биржи Kraken через библиотеку `ccxt`,
сохраняет историю котировок в SQLite, анализирует краткосрочный тренд и
отправляет пользователям уведомления по тем активам, на которые они
подписались.

## Цель

Цель проекта — дать пользователю простой автономный инструмент, который
помогает замечать резкие изменения цены без постоянного ручного просмотра
графиков.

Система не принимает торговые решения за пользователя и не является
инвестиционной рекомендацией. Её задача — собрать данные, обнаружить
потенциально важное изменение и быстро доставить сигнал в Telegram.

## Основные возможности

* сбор актуальных цен и объёмов торгов с Kraken;
* хранение истории котировок в локальной SQLite-базе;
* анализ краткосрочного тренда по истории цен;
* фиксация признака аномальной волатильности;
* регистрация пользователей Telegram;
* управление подписками на торговые пары;
* отправка уведомлений подписчикам при обнаружении аномалии;
* архивирование отправленных уведомлений.

## Целевая аудитория

Проект ориентирован на пользователей, которым важно быстро узнавать о
заметных движениях рынка:

* краткосрочные трейдеры, отслеживающие резкие движения цены;
* крипто-аналитики, которым нужна локальная история котировок;
* пассивные инвесторы, желающие получать сигналы о сильной волатильности.

## Внешние системы

В работе проекта участвуют три ключевые внешние среды:

| Система | Роль |
| --- | --- |
| Kraken | источник рыночных котировок |
| Telegram | канал взаимодействия с пользователем |
| SQLite | локальное хранилище данных проекта |

Kraken используется через `ccxt`, Telegram-бот реализован на `aiogram`, а
SQLite хранит активы, пользователей, подписки, историю цен, результаты анализа
и архив уведомлений.

## Границы проекта

В текущей версии проект фокусируется на локальном запуске и учебной
воспроизводимости. Он не включает личный кабинет, web-интерфейс, управление
реальными сделками, платёжные операции и интеграцию с биржевыми аккаунтами
пользователей.

Запускаемая часть проекта состоит из двух процессов:

* Telegram-бот обрабатывает команды и кнопки пользователя;
* сборщик данных получает котировки, запускает анализ и инициирует уведомления.
